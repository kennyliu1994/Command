How to start Github
register github on website (https://github.com/)
install git for pc (https://git-scm.com/)

Normal Command
$ git --version
show version
$ git status
show status currently
$ git log
list all commit history
$ git diff 
compare file or commit file
$ git show 
show commit content currently

1.upload
$ git config --global user.email "<email>"
$ git config --global user.name "<name>"
set profile who push to remote
$ git config --list

$ git clone <HTTP from Github repository> 
clone to local

$ git init  
create Repository (input git init in right Repository)

$ git add -A 
add modified file to Staging Area

$ git commit -a -m "want to say"
Store to Repository
$ git commit -e 
say more

$ git push
Push to Github(remote)

2.go back to previous version
$ git log <version number>
(ex:5f5bd1...)
$ git reset --hard <version number>

4.go to future version
$ git reflog
$ git reset --hard <version number>
