使用之前
register github on website (https://github.com/)
install git for pc (https://git-scm.com/)

一般指令
$ git --version
版本
$ git status
顯示狀態
$ git log
顯示全部commit的內容
$ git diff 
(ex:diff N9-4.Q /d/QMScript/N9-4.Q)
比較檔案內容
$ git show 
顯示目前commit內容

push到github
$ git config --global user.email "<email>"
$ git config --global user.name "<name>"
設定誰要push
$ git config --list
顯示全部設定

$ git clone <HTTP from Github repository> 
下載到local

$ git init  
建立Repository(輸入git init在當下Repository)

$ git add -A 
加入修改的檔案到暫存區(Staging Area)
$ git add .
只加入修改的檔案

$ git commit -a -m "敘述"
存版本版本庫(Repository)
$ git commit -e 
可敘述更多

$ git push
push到Github(remote)

2.go back to previous version
$ git log <version number>
(ex:5f5bd1...)
$ git reset --hard <version number>

4.go to future version
$ git reflog
$ git reset --hard <version number>
